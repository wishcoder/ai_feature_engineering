Issue Summary

Issue ID: FW004

Component: Firewall

Issue Type: Change

Date Reported: 2024-03-27

Affected Systems: Network Traffic Control

Issue Description

Recent changes to the firewall configuration have inadvertently blocked critical traffic, impacting service communication.

Architecture Overview

Outside network connects to Router, Router connects to firewall that sends traffic to App, App connects to Service A, which connects to Service B, which then connects to Service C (Database).

Possible Causes

  • Incorrectly applied firewall rules.
  • Firewall firmware update introducing unexpected behavior.

Resolution Steps

  1. Review Firewall Configuration: Verify all rules and settings against the intended security policy.
  2. Firewall Logs: Check the logs for denied traffic that should be allowed.
  3. Test Changes: In a controlled environment, test changes before applying them to production.

Resolution Outcome

Reverting the recent firewall rule changes and then carefully reapplying them with the correct configurations resolved the traffic block issue.
